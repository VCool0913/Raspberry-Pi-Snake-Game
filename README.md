# Raspberry-Pi-Snake-Game
This is a snake game designed to work on the raspberry pi. It is meant to work with a 16 x 16 LED board that our class made, a touchscreen tablet, and a controller that we also made.


The output for the LED board is D24

Button pin input:
UP: 5
DOWN: 6
LEFT: 13
RIGHT: 19


The admin window password is "0028"
